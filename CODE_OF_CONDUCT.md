# Code of Conduct

## Our Commitment

我们承诺为所有参与者提供友善、尊重、无骚扰的协作环境，不因经验水平、身份、地区、语言、外貌、能力、职业或观点差异而区别对待。

## Expected Behavior

- 使用清楚、专业和尊重他人的语言。
- 对代码和方案提出具体、可验证的意见，不攻击个人。
- 接受合理的技术质疑，并说明证据和取舍。
- 尊重用户素材、客户信息、肖像、声音和商业数据的隐私。
- 发现安全问题、密钥或个人信息时停止公开传播，并改用私下报告渠道。
- 在付费 API、内容合规和第三方 Provider 问题上保持诚实，不夸大验证状态。

## Unacceptable Behavior

- 骚扰、威胁、歧视、人身攻击或持续性的恶意争论。
- 未经授权公开他人的个人信息、API Key、客户素材或请求记录。
- 故意绕过安全门禁、付费上限、确认流程或内容合规限制。
- 将候选结果、局部测试或第三方服务宣传为已经验证的最终生产能力。
- 发布恶意代码、欺骗性修改或隐藏的数据外传行为。

## Scope

本行为准则适用于仓库、Issue、Pull Request、Discussion、Release，以及代表本项目参与公开交流的场景。

## Reporting and Enforcement

如需私下报告严重行为问题，请使用 GitHub Private Security Advisory，并在标题中注明 `Conduct report`。不要在公开 Issue 中发布当事人的个人信息或私密证据。

项目维护者可以编辑或删除不当内容，关闭讨论，拒绝贡献，或暂时、永久限制违反行为准则的参与者。处理时应尽量保护举报人和相关人员的隐私。
